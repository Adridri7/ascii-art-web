## </> Run Locally

Clone the project repo

```bash
  git clone https://zone01normandie.org/git/arocchet/ascii-art-web.git
```

Go to the project directory

```bash
  cd ascii-art-web/cmd
```

Run the example

```bash
  go run .
```
